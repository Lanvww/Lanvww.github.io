Project description

This project is based on Conway's Game of Life. Besides the original display experience, I added six different controls and allow to user to have more varied experience with the shape, color, fps speed, cell size, and mouse click. If the user chooses a color mode or random, the game and all the elements on the page will change to match the color.

Control instruction

  • Click button to replay or random all the values
  • Change shape by the radio buttons
  • Change FPS by the slider
  • Change Cell Size by the slider
  • Choose color from the select list
  • Mouse click on the Canvas to generate specific size world
